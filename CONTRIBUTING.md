# Contributing

## Zweck

Dieses Dokument beschreibt, wie Beitraege zum Fjölnir Agent Framework vorbereitet, geprueft und eingereicht werden sollen.

## Scope

Der Beitragprozess gilt fuer Dokumentation, Templates, Beispiele, Checklisten und spaetere Tooling-Erweiterungen.

## Struktur

- Aenderungsvorschlag mit klarem Ziel.
- Begruendung und betroffene Framework-Module.
- Review gegen Constitution, Writing Guide und relevante Modi.
- Pull Request mit nachvollziehbarer Beschreibung.
